# Imran Ahsan Academic Website

This version uses `index.html` instead of `index.md` to ensure exact alignment across GitHub Pages themes and screen sizes.

## Update the live site

1. Delete or rename the old `index.md` in the repository.
2. Upload `index.html`, `_config.yml`, and the complete `assets` folder to the repository root.
3. Commit the changes and wait for GitHub Pages to redeploy.

The profile image path is `assets/profile.jpg`.
